## About The Project


BreedingCharts is a free program for easily making charts of plant lineages for plant breeders. 

BreedingCharts includes features such as:

-Creating and managing plant lineages and charts with customization.   
-Exporting charts as .png and .csv.

## Getting Started

Run the precompiled executable .exe for Windows. 



### Installation from source

1. Clone the repo
2. Unzip and run:
   ```sh
   pip install -r requirements.txt
   ```
3. Run with:
   ```sh
   python main.py
   ```
